# ESLyric-LyricsSource

一些用于 ESLyric 的高级歌词源，将 KRC (酷狗)、QRC (QQ 音乐) 和 YRC (网易云音乐) 的逐字歌词和翻译歌词转换为 ESLyric 支持的格式。

**不要**在这里反馈任何与 ESLyric 插件相关的问题，我**不是**其作者，不对其负任何责任！

佛系更新，有问题**肯定不会**及时解决。

## 用哪个版本

## Legacy

如果你的 ESLyric 是老版本，用这里的文件。

仅酷狗支持逐字歌词和翻译歌词。

## Current

新版本 ESlyric 使用这里的文件。

酷狗、QQ 音乐和网易云音乐三个歌词源都支持逐字和翻译歌词。
